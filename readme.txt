== Description ==

Quickly and easily iconize posts, pages, menu items, widget titles, taxonomies, and pretty much everything, using icon picker with plenty of customization options. No shortcodes. No coding skills required. Developer friendly.

Iconize is here to make your life easier, by providing simple and easy to use interface for inserting icons.

Features:

* 8 Optional Icon Fonts included - Dashicons, Font Awesome 4.7, Font Awesome 5 Free (Solid, Regular, Brands), Bootstrap Icons, Foundation 3 Icons and Iconoir
* Upload custom fonts
* The only actual WYSIWYG icons classic editor plugin (works inside Gutenberg classic editor block)
* Menu icons (Customizer support)
* Widget title icons (Customizer support)
* Post Categories, Post Tags, Custom Taxonomy icons
* Effects, color, size, alignment, custom CSS classes options
* 26 CSS transform options
* 13 CSS animation options
* 56 CSS hover effects
* Optionally use included fonts inside popular free page builders (Elementor, Beaver Builder, Page Builder by SiteOrigin)
* CMB2 Custom field

Getting started:

Here's a link to our [Documentation](https://thatplugin.com/docs/iconize/ "Iconize Documentation"), we doubt that you'll need anything else but feel free to ask if you have any question.

== Installation ==

1 a). Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1 b). Activate the plugin through the 'Plugins' screen in WordPress
2. Use the Settings->Iconize screen to configure the plugin
